# 💻 Web Development – Topic Wise Codes

A collection of beginner-friendly web development examples organized by topic to make learning, practicing, and revising front-end concepts simple and effective.

## 📖 About

This repository is a personal learning archive containing hands-on code examples covering the fundamentals of web development. Every topic is separated into its own section, making it easy to revisit concepts and track progress while learning.

* 🎨 CSS
* 📦 CSS Box Model

* 🖥️ CSS Display
* 🧩 CSS Flexbox
* 🔲 CSS Grid

* 🅱️ Bootstrap
* ⚡ JavaScript Basics
* 🌐 Mini Projects & Practice Files
* 🚀 More topics added regularly

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript
* Bootstrap 5
